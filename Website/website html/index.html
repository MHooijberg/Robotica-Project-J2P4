ROBOTICS

Welkom

Welkom op de Robotica website van groep 2, hier kunt u de progressie van ons project volgen. Tijdens dit project bouwen wij aan een slimme robot. Deze robot moet verschillende opdrachten kunnen uitvoeren zoals het volgen van een blauw blokje, een parkour afgaan en dansen op muziek.

Ook moest de robot een maatschappelijk probleem kunnen oplossen. Ons maatschappelijk probleem wat we willen oplossen is dat het in werkplaatsen vaak een bende is en er geen apparaat is die het makkelijk voor je kan opruimen. Hiervoor hebben we onze robot gemaakt die dit wel kan, zodat de werkplaats mooi schoon blijft en het niet heel veel moeite kost om de werkplaats mooi schoon te houden.
